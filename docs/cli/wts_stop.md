## wts stop

Stop worktree process(es)

```
wts stop [worktree] [flags]
```

### Options

```
      --all            stop all worktrees
      --group string   stop active worktree in group
  -h, --help           help for stop
```

### Options inherited from parent commands

      --state string    path to state file (default ~/.workswitch/state.yaml)
