## wts add

Add a worktree directory assignment

```
wts add <dir> [flags]
```

### Options

```
      --group string     optional group override
  -h, --help             help for add
      --name string      worktree alias name
      --process string   process profile name
```

```
      --state string    path to state file (default ~/.workswitch/state.yaml)
```
