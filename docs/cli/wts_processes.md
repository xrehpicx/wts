## wts processes

List process profiles from config

### Synopsis

Show process profile names and commands from .wts.yaml.

```
wts processes [flags]
```

### Examples

```
wts processes
  wts processes --config ../other/.wts.yaml
```

### Options

```
  -h, --help   help for processes
```

### Options inherited from parent commands
